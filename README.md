# MRI Production

Canonical source-of-truth repository for MRI dashboard and operations docs.

## Structure
- `apps/dashboard` — Next.js dashboard app (Vercel deploy root)
- `docs/` — operational and launch planning references

## Local Dev
```bash
cd apps/dashboard
npm install
npm run dev
```

## Deploy
- GitHub: `bina-digital/mri-production`
- Vercel Project: `mri-production`
- Vercel Root Directory: `apps/dashboard`
